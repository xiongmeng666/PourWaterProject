# PourWaterProject - 高性能水排序解谜游戏架构演示

本项目是一款基于 **Cocos Creator 3.x** 开发的经典水排序（Water Sort）解谜游戏。项目核心目标是展示在游戏开发中如何通过严谨的架构设计解决逻辑复杂性，并利用 Shader 提升视觉表现力。

## 核心技术亮点

### 1. 严格的 MVP 架构设计
项目采用 **MVP** 架构，实现了业务逻辑与渲染表现的深度解耦：
* **Model**: 纯 TypeScript 编写，负责水颜色数据管理、核心倒水规则校验及撤回历史栈。独立于引擎，具备极强的可测试性。
* **Presenter**: 逻辑中枢，通过自定义接口 (IGameView) 指挥视图层，并持有全局动画锁防止并发冲突。
* **View**: 仅负责视觉表现与动画播放，通过子视图组件化管理瓶子（BottleView）和结算面板。

### 2. 物理感液体渲染 (Custom Shader)
自定义 Effect 实现液体渲染，支持：
* **动态面水平修正**: 结合瓶身旋转角度（Rotation），实时计算液面水平效果。
* **层级锁定算法**: 解决了倾斜状态下多层颜色交界线偏移的图形学难题。
* **正弦波浪模拟**: 增加动态波浪效果，提升游戏视觉质感。
* **配置驱动**: 颜色层数上限（支持至8层）与瓶口物理边界均通过全局配置与 TypeScript 动态同步。

### 3. 稳健的游戏逻辑
* **100% 可解关卡生成**: 采用逆向推导算法（Reverse Engineering），从满瓶状态反向抽水打乱，确保随机生成的关卡绝对有解。
* **精准倒水逻辑**: 支持“部分倒水”规则，自动计算源瓶同色层数与目标瓶剩余空间的交集。
* **内存与性能优化**: 严格管理生命周期，在 `onDestroy` 中执行全方位的补间动画（Tween）停止、定时器清理及闭包解绑，防止内存泄漏。

## 环境要求
* **引擎版本**: Cocos Creator 3.8.8
* **编程语言**: TypeScript
